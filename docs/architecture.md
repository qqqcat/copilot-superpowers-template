# Architecture

## Overview
This repository is a reusable Copilot Agent superpowers template.

## Layers
- Agent entry and context (`.github/copilot-agent.md`, `.github/project-context.md`)
- Instruction hierarchy (`.github/instructions/**`)
- Automation workflow (`.github/workflows/**`)
- Local editor defaults (`.vscode/settings.json`)

## Goals
- High-quality autonomous engineering behavior
- Consistent planning, implementation, testing, and review flow
- Reusable template for new projects
